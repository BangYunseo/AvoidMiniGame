# Avoid-Mini-Game
Bullet Avoid Mini Game(가제 : Fish.io)
* 총알 피하기 미니 게임 프로젝트

#### 추가할 사항

* 오브젝트 추가 및 수정
  * 쉴드 오브젝트
    * 데미지 1회 방어
    * 간간히 생기게 하기
  * Player
    * Player가 Food를 먹어서 커질수록 메인 카메라 조정 필요
* 배경 추가
  * 배경의 디자인 추가 필요 (바다)
  * 배경을 좀 넓게 잡고 끝부분을 선으로 표시해두거나 배경의 디자인을 물로 바꾼 후 어항으로 표시해도 괜찮을 것 같음
* RECORD 기능 추가
* QUIT 기능 추가
* 게임 멈추는 기능 추가
