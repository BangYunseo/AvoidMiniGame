# Avoid-Mini-Game
Bullet Avoid Mini Game
* 총알 피하기 미니 게임 프로젝트

#### 추가할 사항
* 다른 오브젝트 추가 및 수정
  * Food 오브젝트를 정상적으로 변환
    * Food 오브젝트와 충돌할 시 Food 오브젝트가 사라지고 게임오버는 되지 않도록 변환
    * 체력 시스템을 도입하여 Food 오브젝트가 HP를 추가해주는 방안으로 변환
    * Food 오브젝트 크기 수정
* 배경 추가
